# Aryap - Workflow

## Gelistirme Sureci
1. Task-index'ten siradaki task'i al
2. Ilgili dosyalari olustur/duzenle
3. `pnpm dev` ile test et
4. Task durumunu guncelle
5. Session notlarini guncelle

## Branch Stratejisi
- `main` — Production-ready kod
- `dev` — Gelistirme branch'i
- Feature branch'leri gerektiginde

## Commit Formati
```
type(scope): description
```
Types: feat, fix, style, refactor, docs, chore, perf
