# Git Full - Aryap Projesi

Tum degisiklikleri commit'le ve push et:

1. `git status` ile degisiklikleri kontrol et
2. `git diff --stat` ile ozet goster
3. Anlamli bir commit mesaji olustur (Conventional Commits formati)
4. `git add .` ve `git commit`
5. `git push` (eger remote varsa)
6. `aryap-docs/CHANGELOG.md` dosyasini guncelle

Commit mesaji formati:
```
type(scope): description

- Detail 1
- Detail 2
```

Type'lar: feat, fix, style, refactor, docs, chore, perf
